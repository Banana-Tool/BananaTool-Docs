## 参数说明  <a href="#can-shu-shuo-ming" id="can-shu-shuo-ming"></a>

1. **前缀** : 生成的靓号地址的开头号码
2. **后缀** : 生成的靓号地址的尾部号码
3. **包含** : 生成的靓号地址中包含的号码
4. **线程数** : 开启多线程后,使用设备性能越好,生成速度越快,
